# 🧠 Medical MRI Image Analyzer

A Python-based medical image processing project developed using OpenCV and Google Colab to analyze MRI images. The application enhances medical images through grayscale conversion, histogram analysis, and histogram equalization techniques.

## :sparkles: Features

- Load MRI images
- Convert images to grayscale
- Generate image histograms
- Apply histogram equalization
- Enhance image quality
- Visualize processing results

## 🛠 Technologies

- Python
- OpenCV
- NumPy
- Matplotlib
- Google Colab

## :rocket: Getting Started

Install the required libraries:

bash
pip install opencv-python numpy matplotlib


Run the project:

bash
python main.py


## :dart: Project Goal

This project was developed to explore medical image processing techniques and strengthen practical skills in Computer Vision, Python, and Healthcare AI.
